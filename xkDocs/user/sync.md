# 课程信息同步

当每次课程更新后，会自动进行数据的同步。如果没有数据变更，同步是无感的。数据变更有下面几种情况。

## 信息变更

![课程信息变更](https://static.xialing.icu/img/202601251454981.webp)

如果课程的开课时间、开课教师、教室等信息发生变更，会提示用户，并询问是否同步。

## 课程关停

![已关课](https://static.xialing.icu/img/202601251510828.webp)

如果选择的课程关停，同步后会清除本课程。

## 课程冲突

![课程冲突](https://static.xialing.icu/img/202601251513260.webp)

如果课程冲突，会把冲突的两门课程移动到备选状态。
